# Colegio Andino - Sitio Web

Sitio web del Colegio Andino desarrollado con HTML, CSS y Bootstrap.

## Características

- Diseño responsive
- Fuente Gilroy personalizada
- Colores institucionales del Colegio Andino
- Múltiples páginas: Inicio, Acerca de, Cursos, Contacto, Equipo, Testimonios

## Estructura del Proyecto

```
elearning-1.0.0/
├── index.html          # Página principal
├── about.html          # Acerca de
├── courses.html        # Cursos
├── contact.html        # Contacto
├── team.html           # Equipo
├── testimonial.html    # Testimonios
├── css/                # Estilos
│   ├── bootstrap.min.css
│   ├── style.css
│   └── gilroy-fonts.css
├── fonts/              # Fuentes Gilroy
├── img/                # Imágenes
└── js/                 # JavaScript

```

## Colores

- **Primario**: `#ae0c26` (Rojo - Menú y botones)
- **Oscuro**: `#1a1a2e` (Azul oscuro - Footer)
- **Claro**: `#f0f4f8` (Gris azulado claro)

## Despliegue

Este proyecto puede desplegarse en:
- GitHub Pages
- Netlify
- Vercel
- Cualquier servidor web estático
